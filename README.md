<h2>📌 Descrição do Projeto</h2>
<p>Este software foi desenvolvido durante o meu terceiro ano do ensino médio para a disciplina de "Programação Orientada a Objetos II" no qual simula uma Petshop sendo possível realizar ações de edição, consulta, exclusão e cadastramento de agendamentos.</p>

## 🚀 Como Executar o Projeto
Para executar o projeto, siga os passos abaixo:
1. Abra o **Apache NetBeans**.
2. Clique em **File → Open Project**.
3. Selecione a pasta onde o projeto está salvo.
4. No painel lateral, localize o projeto.
5. Clique com o botão direito sobre o projeto.
6. Selecione **Run** ou pressione **F6**.
</ol>
